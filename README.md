一个简洁，使用 **Jetpack Compose** 构建的 Android 音乐播放器。



---

## 特性

* 🎵 本地音乐播放
* ☁️ 在线音乐支持（基于 [TuneFreeNext](https://tunefree.fun/)）
* 📱 多设备支持：手机 / 平板 / Wear OS
* 🧩 纯 Kotlin + Jetpack Compose UI
